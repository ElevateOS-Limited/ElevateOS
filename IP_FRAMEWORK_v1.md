# IP_FRAMEWORK_v1

Status: Draft framework for founder approval before incorporation filing.

## Option A — Assignment-first
- All project IP (code, brand, data pipeline, internal docs) is assigned to the company at formation.
- Founders receive equity consideration and any pre-agreed compensation for pre-existing IP.
- Pros: clean diligence, simpler investor narrative.
- Risk: harder to unwind if company setup is delayed.

## Option B — License-first (temporary) [RECOMMENDED FOR CURRENT STAGE]
- Pre-incorporation, creator(s) grant exclusive license to operating team/company-in-formation.
- At incorporation trigger, IP assignment executes automatically under pre-agreed terms.
- Compensation for pre-existing IP is fixed in writing (no later renegotiation).
- Pros: practical while minors/family governance and entity setup are in progress; preserves continuity.

## Selected default for this phase
**Option B (License-first temporary), with automatic assignment trigger at incorporation.**

## Mandatory clauses (both options)
1. IP inventory schedule (what exists now).
2. Assignment scope (source code, content, prompts, datasets, trademarks, domain assets).
3. Moral rights waiver where legally permitted.
4. Confidentiality and non-disclosure.
5. Dispute venue and governing law (to be finalized with counsel).
